Vet Front es una aplicación web de frontend desarrollada con HTML, CSS y JavaScript que simula el sitio de una clínica veterinaria.

El proyecto está enfocado en ofrecer una interfaz sencilla, clara y fácil de usar, donde se presenta información básica sobre la veterinaria y sus servicios. Su objetivo principal es practicar el diseño y la estructura de una aplicación web sin utilizar frameworks ni backend, centrándose únicamente en la experiencia visual y en interacciones básicas del usuario.
